# Smart-Face-Attendance

Smart-Face-Attendance is an intelligent attendance management system that uses facial recognition technology to automate the process of marking attendance. This system is designed to save time, reduce manual errors, and improve accuracy in recording attendance for classrooms, offices, and other institutions.



## Features
- Automated attendance marking using facial recognition.
- Real-time detection of faces.
- Storage of attendance data in CSV or database format.
- Support for multiple users.
- User-friendly interface.
- Option to generate attendance reports.

## Technologies Used
- Python
- OpenCV (Computer Vision)
- face_recognition library
- NumPy
- Pandas
