## Overview
An NLP-based fitness chatbot built using Python, NLTK, TensorFlow, and Streamlit.
The chatbot provides workout, diet, recovery, and fitness guidance using intent classification.

## Features
- NLP preprocessing (tokenization, lemmatization)
- Bag-of-Words model
- TensorFlow neural network
- Confidence-based response handling
- Interactive Streamlit UI

## Tech Stack
- Python
- NLTK
- TensorFlow / Keras
- Streamlit

pip install -r requirements.txt
